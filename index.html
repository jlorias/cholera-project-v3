Frameworks:

The majority of this work was done using the following:

  • plot.ly
  • HTML, CSS, and Javascript
  • semantic UI

The purpose of this project is to showcase datasets of Cholera deaths and attacks of affected males and females in the UK and Naples. This data was transformed into visual representations such as bar, pie, and line charts, and maps.
